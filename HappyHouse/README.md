# HappyHouse
- 이사를 하고자 할 동네에서 원하는 아파트, 주택을 찾아낼 목적으로 매매/전월세 거래 정도를 Web페이지를 통해 보여준다.
- 사용자의 동작에 따라 검색 지역 실시간 날씨 정보, 금액별 정보 조회 서비스를 추가로 제공한다.
- 이메일 인증을 통한 회원가입으로 신뢰성을 높인다.
- 비회원/회원, 관리자/유저 에 따른 동작 제한이 있다.
